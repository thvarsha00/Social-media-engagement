## 📌 Overview

This project analyzes ~12,000 social media posts across brands and campaigns to answer a simple business question: **can we group posts into meaningful engagement segments, and predict which segment a new post belongs to?**

It combines unsupervised learning (to discover the segments) with supervised learning (to predict them), and ships as a working Streamlit app rather than a static notebook.

## 🧠 Approach

1. **EDA & Preprocessing** — cleaned and engineered features from likes, shares, comments, impressions, sentiment, hashtags, mentions, and campaign metadata
2. **NLP Analysis** — sentiment/emotion scoring, TF-IDF keyword extraction, topic modeling (LDA)
3. **Unsupervised Learning** — compared 4 clustering methods (K-Means, Hierarchical, DBSCAN, Mean Shift) on PCA-reduced features; **K-Means (k=4)** was carried forward
4. **Supervised Learning** — trained 4 classifiers to predict the K-Means segment from 7 engagement metrics

## 📈 Results

| Model | Accuracy | ROC-AUC |
|---|---|---|
| Logistic Regression | 0.87 | 0.980 |
| SVM (rbf) | 0.95 | 0.991 |
| Random Forest | 0.98 | — |
| **Gradient Boosting** ⭐ | **0.98** | **0.9996** |

**Gradient Boosting** was selected for deployment based on the strongest combined accuracy and ROC-AUC.

The 4 discovered engagement segments (labeled dynamically by rank, not hardcoded):
- 🔴 Low Engagement
- 🟠 Moderate Engagement
- 🟢 High Engagement
- 🟣 Viral / Outlier

## 🖥️ The App

Built with Streamlit, the app has three views:

| Tab | What it does |
|---|---|
| 🔮 **Predict Single** | Enter a post's metrics manually, get its predicted segment + confidence score, with a chart comparing it to the segment average |
| 📁 **Batch Predict** | Upload any CSV of posts — get predictions for every row, plus a downloadable results file |
| 📊 **Dashboard** | Explore segment sizes, average engagement rate, and feature comparisons across all 4 segments |

## 🛠️ Tech Stack

- **Analysis:** Python, pandas, scikit-learn, NLTK/spaCy (NLP), matplotlib/seaborn
- **Modeling:** K-Means, DBSCAN, Hierarchical Clustering, Mean Shift, Logistic Regression, SVM, Random Forest, Gradient Boosting
- **App:** Streamlit, Plotly
- **Deployment:** Streamlit Community Cloud

## 🚀 Running Locally

```bash
git clone https://github.com/thvarsha00/Social-media-engagement.git
cd Social-media-engagement

python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # macOS/Linux

pip install -r requirements.txt
streamlit run app.py
```

The app expects `engagement_cluster_classifier.pkl` and `cluster_profile.csv` in the same folder as `app.py` (both are generated by the export cell at the end of the notebook).
